# data/

스크래핑 데이터 및 그래프 export 저장소.

## 하위 디렉토리

| 디렉토리 | 역할 |
|----------|------|
| `products/대출/` | 대출 상품 MD 파일 40개 (신용대출, 담보대출, 전월세대출, 자동차대출) |
| `products/예금/` | 예금 상품 MD 파일 14개 (정기예금, 입출금통장) |
| `products/적금/` | 적금 상품 MD 파일 12개 (적금, 청약) |
| `graph/` | D3.js용 graph.json export (470 노드, 712 링크) |
| `raw/` | 스크래핑 원본 데이터 (현재 비어있음) |
