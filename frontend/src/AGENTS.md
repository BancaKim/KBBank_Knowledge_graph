# frontend/src/

React 앱 소스코드.

## 파일

| 파일 | 역할 |
|------|------|
| `App.tsx` | 메인 컴포넌트 - ResizablePanels(채팅+그래프), 필터 상태, 하이라이트 관리 |
| `main.tsx` | React DOM 엔트리포인트 |
| `config.ts` | API Base URL 설정 (환경변수 지원) |
| `App.css` | 전역 스타일 |
| `index.css` | 기본 CSS 리셋 |
