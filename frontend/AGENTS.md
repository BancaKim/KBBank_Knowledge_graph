# frontend/

React + TypeScript + D3.js 프론트엔드. Vite 빌드.

## 파일

| 파일 | 역할 |
|------|------|
| `package.json` | 의존성 (react 19, d3 7, typescript) |
| `vite.config.ts` | Vite 빌드 설정 |
| `tsconfig.json` | TypeScript 설정 (프로젝트 참조 루트) |
| `tsconfig.app.json` | 앱 소스코드용 TypeScript 설정 |
| `tsconfig.node.json` | Vite 설정 파일용 TypeScript 설정 |
| `index.html` | HTML 엔트리포인트 (Noto Sans KR 폰트 로드) |
| `eslint.config.js` | ESLint 설정 |
