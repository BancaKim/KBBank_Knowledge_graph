# 신생아 특례 디딤돌대출 (주택구입)

2년 내 출산(입양) 가구를 위한 초저금리 주택구입 대출. 디딤돌대출 기반이나 소득·주택가격 조건이 대폭 완화.

## 디딤돌 vs 신생아 특례 핵심 차이

| 항목 | 디딤돌대출 | 신생아 특례 |
|------|----------|-----------|
| **소득** | 6천만원 | **1.3억원** (맞벌이 2억) |
| **주택가격** | 5억원 | **9억원** |
| **최대한도** | 3.2억원 | **4억원** |
| **최저금리** | 2.55% | **1.80%** |
| **추가 출산 우대** | 없음 | **0.2%p + 5년 연장** |

## 자격요건

| 항목 | 조건 |
|------|------|
| **출산요건** | 대출접수일 기준 **2년 내 출산·입양** (2023.1.1 이후 출생아) |
| **소득** | 부부합산 연소득 **1.3억원 이하** |
| | 맞벌이: **2억원 이하** (부부 각 1.3억원 이하) |
| **자산** | 순자산가액 **5.11억원** 이하 |
| **주택가격** | 담보주택 평가액 **9억원** 이하 |
| **면적** | 전용 85㎡ 이하 (읍·면 100㎡) |
| **주택보유** | 세대원 전원 무주택 |
| **신용** | 연체·부도 이력 없음 |

## 대출한도 · LTV · DTI

| 항목 | 조건 |
|------|------|
| **최대한도** | **4억원** |
| **LTV** | 70% (생애최초 80%, 규제지역 70%) |
| **DTI** | 60% |

| 부부합산 연소득 | 10년 | 15년 | 20년 | 30년 |
|---------------|:----:|:----:|:----:|:----:|
| ~2천만원 | **1.80%** | 1.90% | 2.00% | 2.05% |

| 8.5천만~1억원 | 2.90% | 3.00% | 3.10% | 3.20% |

> **지방 소재 주택**: 위 금리에서 **0.2%p 인하**

## 특례금리 적용기간 · 추가출산 우대

```
기본 특례금리 적용: 5년

추가 출산 시:
  · 금리 우대: 자녀 1명당 -0.2%p
  · 기간 연장: 자녀 1명당 +5년
  · 최장 적용: 15년

예시: 1자녀 출산 후 대출 → 5년 특례 적용
      2년 후 둘째 출산 → 0.2%p 추가 인하 + 5년 연장 = 총 10년 특례
      이후 셋째 출산 → 0.2%p 추가 인하 + 5년 연장 = 총 15년 특례
```

> 특례기간 종료 후: 디딤돌대출 일반금리로 전환

## 전자계약 우대

- 부동산 전자계약 체결 시 **0.1%p 추가 인하** (2026.12.31까지 접수분)

## 대환대출

| 항목 | 조건 |
|------|------|
| 대상 | 기존 주담대 → 신생아특례로 전환 |
| 주택보유 | **1주택 세대주도 가능** |
| 소득기준 | 동일 (1.3억원 이하) |
| 신청기한 | 제한 없음 |

## 대출기간 · 상환방식

- **기간**: 10, 15, 20, 30년
- **거치**: 비거치 또는 1년 거치
- **상환**: 원리금균등 / 원금균등 / 체증식 분할상환

## 자격 판별 흐름

```
1. 2년 내 출산/입양 여부 확인
   → NO: 일반 디딤돌대출 안내

2. 부부합산 소득 확인
   → 1.3억 초과 (맞벌이 2억 초과): 대상 아님
   → 6천만 이하: 신생아특례 + 일반 디딤돌 모두 가능 → 금리 비교

3. 주택가격 확인
   → 9억 초과: 대상 아님
   → 5억 초과~9억: 신생아특례만 가능 (디딤돌 불가)
   → 5억 이하: 둘 다 가능 → 금리 비교

4. 금리 비교 판단
   · 소득 6천만 이하 + 주택 5억 이하: 두 상품 금리 비교하여 유리한 쪽
   · 소득 6천만 초과 OR 주택 5억 초과: 신생아 특례만 가능
```
