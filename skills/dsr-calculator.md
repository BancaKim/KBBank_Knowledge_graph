# DSR (총부채원리금상환비율) 계산기

금감원 DSR 산정기준을 정확히 반영하여 대출 가능 여부와 최대 대출 가능액을 산출합니다.

## 주요 기능

### 1. DSR 계산 (`calculate_dsr`)
신규 주택담보대출의 DSR을 계산하고 규제 충족 여부를 판단합니다.

**필수 입력**:
- `annual_income`: 연소득 (만원 단위)
- `new_loan_amount`: 신규 대출 원금 (만원 단위)
- `new_loan_rate`: 약정 연이자율 (%)
- `new_loan_months`: 상환기간 (개월)

**선택 입력**:
- `new_loan_method`: 상환방식 (원리금균등/원금균등/만기일시, 기본: 원리금균등)
- `new_loan_rate_type`: 금리유형 (변동/혼합5년/혼합10년/고정 등, 기본: 변동)
- `region`: 지역 (수도권/지방/규제지역, 기본: 수도권)
- `sector`: 금융권 (은행/2금융, 기본: 은행)
- `existing_loans`: 기존 대출 (형식: `잔액:금리:잔여개월:상환방식:유형:금리유형`, `|`로 구분)

### 2. 최대 대출 가능액 (`calculate_max_mortgage_by_dsr`)
DSR 한도 내에서 받을 수 있는 최대 대출 금액을 역산합니다.

**출력 포함 사항**:
- 상환방식별 최대 대출 가능액 비교표
- 상환기간별(10~40년) 최대 대출 가능액 비교표
- 금리유형별(변동/혼합/고정) 최대 대출 가능액 비교표

## 적용 규제 기준 (금감원 DSR 산정기준)

### DSR 한도
| 금융권 | DSR 한도 | 적용 기준 |
|--------|:-------:|----------|
| 은행권 | **40%** | 총 대출잔액 1억 초과 |
| 2금융권 | **50%** | 총 대출잔액 1억 초과 |

### 스트레스 가산금리 (3단계, 2025.7.1~)
| 구분 | 기본 가산금리 |
|------|:----------:|
| 수도권/규제지역 주담대 | +3.0%p |
| 지방 주담대 | +0.75%p |
| 신용대출 (1억 초과) | +1.5%p |

### 금리유형별 스트레스 적용비율
| 금리유형 | 적용비율 | 수도권 실제 가산 |
|---------|:------:|:-----------:|
| 변동금리 (5년 미만) | 100% | +3.0%p |
| 혼합형 (고정 5~9년) | 80% | +2.4%p |
| 혼합형 (고정 9~15년) | 60% | +1.8%p |
| 혼합형 (고정 15~21년) | 40% | +1.2%p |
| 순수 고정금리 | 0% | +0.0%p |

### 만기일시상환 DSR 산정 (금감원 기준)
만기일시상환은 **원리금균등 전환(의제)** 하여 산정하며, 대출유형별 **산정만기 상한** 적용:

| 대출 유형 | 산정만기 상한 |
|----------|:----------:|
| 주택담보대출 | 10년 |
| 신용대출 (일시상환) | 5년 |
| 마이너스통장/한도대출 | 10년 |
| 오피스텔 담보대출 | 8년 |
| 전세보증금 담보대출 | 4년 |
| 카드론 | 3년 |

### 기존대출 처리
- 기존대출은 **약정금리** 기준으로 연상환액 산정
- 신규 강화된 스트레스 금리가 기존대출에 **소급 적용되지 않음** (금융위 규정)

## 사용 예시

```
# 연소득 5천만원, 3억 대출, 금리 4.5%, 30년 원리금균등, 변동금리, 수도권
calculate_dsr(5000, 30000, 4.5, 360, "원리금균등", "변동", "수도권", "은행")

# 고정금리로 받을 때 (스트레스 가산 0%)
calculate_dsr(5000, 30000, 4.5, 360, "원리금균등", "고정", "수도권", "은행")

# 기존 신용대출 있을 때
calculate_dsr(5000, 30000, 4.5, 360, "원리금균등", "변동", "수도권", "은행",
              "5000:5.0:48:원리금균등:신용대출:변동")

# 최대 대출 가능액 산출
calculate_max_mortgage_by_dsr(5000, 4.5, 360, "원리금균등", "변동", "수도권", "은행")
```

## DSR 산정 공식

```
DSR(%) = (모든 금융부채의 연간 원리금 상환액 합계) / 연간 소득 × 100

포함: 주담대, 신용대출, 자동차할부/리스, 카드론, 현금서비스
제외: 보증부 전세대출, 학자금대출(ICL), 500만원 이하 소액대출
```

### 상환방식별 연간 상환액 산정
- **원리금균등**: M = P × r × (1+r)^n / [(1+r)^n - 1], 연상환 = M × 12
- **원금균등**: 첫해 12개월 합산 (DSR은 최대 상환액 기준)
- **만기일시**: **원리금균등으로 전환**, 산정만기 = min(실제만기, 대출유형별 상한)

### 최대 대출 가능액 역산
- 원리금균등: closed-form 역산 (P = M × [(1+r)^n - 1] / [r × (1+r)^n])
- 원금균등/만기일시: 이진탐색으로 원금 역산
