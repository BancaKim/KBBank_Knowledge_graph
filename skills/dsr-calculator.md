# DSR (총부채원리금상환비율) 계산기

주택담보대출 시 DSR 규제를 정확히 반영하여 대출 가능 여부와 최대 대출 가능액을 산출합니다.

## 주요 기능

### 1. DSR 계산 (`calculate_dsr`)
신규 주택담보대출의 DSR을 계산하고 규제 충족 여부를 판단합니다.

**필수 입력**:
- `annual_income`: 연소득 (만원 단위)
- `new_loan_amount`: 신규 대출 원금 (만원 단위)
- `new_loan_rate`: 연이자율 (%)
- `new_loan_months`: 상환기간 (개월)

**선택 입력**:
- `new_loan_method`: 상환방식 (원리금균등/원금균등/만기일시, 기본: 원리금균등)
- `region`: 지역 (수도권/지방/규제지역, 기본: 수도권)
- `sector`: 금융권 (은행/2금융, 기본: 은행)
- `existing_loans`: 기존 대출 (형식: `잔액:금리:잔여개월:상환방식:유형`, `|`로 구분)

### 2. 최대 대출 가능액 (`calculate_max_mortgage_by_dsr`)
DSR 한도 내에서 받을 수 있는 최대 대출 금액을 역산합니다.

**출력 포함 사항**:
- 상환방식별 최대 대출 가능액 비교표
- 상환기간별(10~40년) 최대 대출 가능액 비교표

## 적용 규제 기준

| 항목 | 기준 |
|------|------|
| DSR 한도 | 은행 40%, 2금융 50% |
| 스트레스 가산금리 (수도권/규제지역) | +3.0%p |
| 스트레스 가산금리 (지방) | +0.75%p |
| 스트레스 가산금리 (신용대출) | +1.5%p |
| 근거 | 스트레스 DSR 3단계 (2025.7.1~), 10·15 대책 |

## 사용 예시

```
# 연소득 5천만원, 3억 대출, 금리 4.5%, 30년 원리금균등, 수도권
calculate_dsr(5000, 30000, 4.5, 360, "원리금균등", "수도권", "은행")

# 기존 신용대출 5천만원 있는 경우
calculate_dsr(5000, 30000, 4.5, 360, "원리금균등", "수도권", "은행",
              "5000:5.0:48:원리금균등:신용대출")

# 최대 대출 가능액 산출
calculate_max_mortgage_by_dsr(5000, 4.5, 360, "원리금균등", "수도권", "은행")
```

## DSR 산정 방법

```
DSR = (모든 부채의 연간 원리금 상환액) / 연소득 × 100

포함: 주담대, 신용대출, 자동차할부/리스, 카드론
제외: 보증부 전세대출, 학자금대출(ICL), 500만원 이하 소액대출
```

### 상환방식별 연간 상환액 산정
- **원리금균등**: 표준 원리금균등 공식
- **원금균등**: 첫해 기준 (최대 상환액)
- **만기일시**: 연이자 + (원금/대출기간) — 금감원 기준 원금균등분할 가정
