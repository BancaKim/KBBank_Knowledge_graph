# 신생아 특례 버팀목 전세대출

2년 내 출산(입양) 가구를 위한 초저금리 전세자금 대출. 버팀목 기반이나 소득·한도 대폭 완화.

## 버팀목 vs 신생아 특례 핵심 차이

| 항목 | 버팀목 (신혼) | 신생아 특례 |
|------|-------------|-----------|
| **소득** | 7,500만원 | **1.3억원** (맞벌이 2억) |
| **한도** | 2.2억원 | **2.4억원** (2025.6.27 이후 3억) |
| **최저금리** | 1.9% | **1.30%** |
| **추가출산 우대** | 없음 | **0.2%p + 4년 연장** |

## 자격요건

| 항목 | 조건 |
|------|------|
| **출산요건** | 대출신청일 기준 **2년 내 출산·입양** (2023.1.1 이후 출생아) |
| **소득** | 부부합산 연소득 **1.3억원 이하** |
| | 맞벌이: **2억원 이하** (부부 각 1.3억원 이하) |
| **자산** | 순자산가액 **3.45억원** 이하 |
| **임차보증금** | 수도권 **5억원** / 수도권 외 **4억원** 이내 |
| **면적** | 전용 85㎡ 이하 (읍·면 100㎡) |
| **주택보유** | 세대원 전원 무주택 |

## 대출한도

| 시기 | 최대한도 | 보증금 비율 |
|------|:-------:|:----------:|
| ~2025.6.26 계약 | **2.4억원** | 80% |
| 2025.6.27~ 계약 | **3.0억원** | 80% |

## 금리 (특례금리)

| 부부합산 연소득 | 금리 범위 |
|---------------|:--------:|
| 2천만원 이하 | **1.30%~** |
| 2천~4천만원 | 1.60%~ |
| 4천~6천만원 | 1.90%~ |
| 6천~8.5천만원 | 2.20%~ |
| 8.5천만~1억원 | 2.50%~ |
| 1억~1.3억원 | 2.80%~ |
| 맞벌이 1.3억~2억원 | 3.10%~4.30% |

> **지방 소재 주택**: 위 금리에서 **0.2%p 인하**

## 특례금리 적용기간 · 추가출산 우대

```
기본 특례금리 적용: 4년

추가 출산 시:
  · 금리 우대: 자녀 1명당 -0.2%p
  · 기간 연장: 자녀 1명당 +4년
  · 최장 적용: 12년

예시:
  1자녀 출산 후 대출    → 4년 특례
  2년 후 둘째 출산      → 0.2%p 인하 + 4년 연장 = 총 8년 특례
  이후 셋째 출산        → 0.2%p 추가 인하 + 4년 연장 = 총 12년 특례
```

> 특례기간 종료 후: 버팀목 일반금리로 전환

## 대출기간 · 상환

- **기간**: 2년 (임차종료일 초과 불가)
- **연장**: 2년 단위, **최장 10년**
- **상환**: 만기 일시상환

## 구입대출(디딤돌)과 비교

| 항목 | 신생아 특례 전세 | 신생아 특례 구입 |
|------|----------------|----------------|
| 용도 | 전세보증금 | 주택구입 |
| 한도 | 2.4~3억원 | 4억원 |
| 특례기간 | **4년** (연장 4년씩) | **5년** (연장 5년씩) |
| 최장특례 | 12년 | 15년 |
| 자산기준 | 3.45억원 | 5.11억원 |
| 보증금/가격 | 수도권 5억/비수도권 4억 | 9억원 이하 |

## 주의사항

1. **전입 의무**: 대출 후 1개월 내 전입, **1년 이상 실거주**
2. **기금대출 중복 불가**: 디딤돌대출과 동시 이용 불가
3. **보증서 필요**: HUG 또는 HF 전세보증서
4. **출산 증빙**: 출생증명서 또는 가족관계증명서
5. **미혼 출산**: 가족관계증명서상 부모 기준으로 소득 심사

## 자격 판별 흐름

```
1. 2년 내 출산/입양 확인
   → NO: 일반 버팀목(청년/신혼/일반) 안내

2. 소득 확인
   → 1.3억 초과 (맞벌이 2억 초과): 대상 아님 → 시중은행 전세대출
   → 7,500만 이하: 신생아특례 + 신혼버팀목 모두 가능 → 금리 비교
   → 7,500만~1.3억: 신생아 특례만 가능

3. 자산 확인
   → 3.45억 초과: 대상 아님

4. 임차보증금 확인
   → 수도권 5억 / 비수도권 4억 초과: 대상 아님

5. 신생아특례 vs 신혼버팀목 금리 비교
   · 소득 5천만 이하: 신생아특례가 훨씬 유리 (1.3%~ vs 1.9%~)
   · 소득 5천~7.5천만: 신생아특례만 가능
```
