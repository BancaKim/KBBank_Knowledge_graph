# 상품 비교

## 트리거
- 사용자가 두 상품 비교, "A vs B", "어떤 상품이 좋을까" 요청 시

## 동작

1. **상품 식별**: 사용자가 언급한 상품명으로 검색
2. **데이터 로드**: MD 파일 또는 Neo4j에서 상품 정보 조회
3. **비교 테이블 생성**

## 비교 항목

| 항목 | 설명 |
|------|------|
| 상품명 | 정식 상품명 |
| 카테고리 | 예금/적금/대출/펀드/신탁/ISA |
| 기본금리 | 최저 금리 |
| 최고금리 | 우대 조건 포함 최고 금리 |
| 금리유형 | 고정/변동 |
| 가입기간 | 최소~최대 기간 |
| 가입금액 | 최소~최대 금액 |
| 가입대상 | 자격 요건 |
| 세제혜택 | 비과세/세금우대 여부 |
| 위험등급 | 1~6등급 |

## 출력 형식
마크다운 테이블로 비교 결과를 제시하고, 각 상품의 장단점을 요약한다.
