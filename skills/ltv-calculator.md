# LTV (주택담보대출비율) 계산기

주택 담보가치 대비 최대 대출 가능액을 산출합니다. 2025.10.15 대책 및 가격대별 절대한도를 반영합니다.

## 산정 공식

```
LTV한도 대출액 = 주택감정가 × LTV한도율 - 선순위채권 - 소액임차보증금

최종 한도 = min(LTV한도 대출액, 가격대별 절대한도)
```

## LTV 한도율

### 규제지역 (투기과열지구·조정대상지역)

2025.10.15 대책으로 서울 전 25개구 + 경기 12개 지역 지정.

| 차주 유형 | LTV 한도 | 비고 |
|----------|:-------:|------|
| 무주택자 | **40%** | |
| 생애최초 주택구매 | **70%** | 6개월 내 전입 의무 |
| 1주택자 (6개월 내 처분) | **50%** | 기존주택 처분조건부 |
| 1주택자 (처분 미이행) | **0%** | 대출 불가 |
| 다주택자 (2채+) | **0%** | 대출 불가 |

### 비규제지역

| 차주 유형 | LTV 한도 |
|----------|:-------:|
| 무주택자 | **70%** |
| 생애최초 | **80%** |
| 1주택자 (처분조건) | **70%** |
| 다주택자 | **60%** |

### 9억 초과 고가주택 차등 (투기과열지구)

| 구간 | LTV |
|------|:---:|
| 9억 이하 | 40% |
| 9억 초과분 | 20% |

> 예: 12억 아파트 → 9억 × 40% + 3억 × 20% = 3.6억 + 0.6억 = **4.2억**

## 가격대별 절대한도 (10.15 대책, 규제지역)

| 주택 시가 | 절대한도 |
|----------|:-------:|
| 15억 이하 | **6억원** |
| 15억 초과 ~ 25억 이하 | **4억원** |
| 25억 초과 | **2억원** |

> LTV로 산출한 금액이 절대한도를 초과하면 절대한도가 적용됩니다.

## 소액임차보증금 공제

LTV 산정 시 담보주택에 설정된 소액임차보증금을 차감합니다.

| 지역 | 소액임차보증금 |
|------|:----------:|
| 서울 | 5,500만원 |
| 수도권 과밀억제권역 | 4,800만원 |
| 광역시 | 2,800만원 |
| 그 외 | 2,000만원 |

## 주요 기능

### 1. LTV 기준 대출한도 계산 (`calculate_ltv_limit`)

**필수 입력**:
- `property_value`: 주택 감정가액 (만원 단위)
- `region_type`: 지역유형 (투기과열/조정대상/비규제)
- `borrower_type`: 차주유형 (무주택/생애최초/1주택처분/다주택)

**선택 입력**:
- `senior_liens`: 선순위채권 잔액 (만원, 기본: 0)
- `deposit_region`: 소액임차보증금 지역 (서울/수도권과밀/광역시/기타, 기본: 서울)
- `has_tenant`: 임차인 유무 (기본: false)

### 2. 출력 내용

- LTV 적용률 (%)
- LTV 기준 대출한도 (만원)
- 가격대별 절대한도 적용 여부
- 최종 대출한도
- 적용된 규제 근거

## 사용 예시

```
# 서울 7억 아파트, 무주택자
calculate_ltv_limit(70000, "투기과열", "무주택")
# → LTV 40% = 2.8억, 절대한도 6억 → 최종 2.8억

# 서울 20억 아파트, 생애최초
calculate_ltv_limit(200000, "투기과열", "생애최초")
# → LTV 70% = 14억, 절대한도 4억 → 최종 4억

# 비규제지역 5억, 1주택 처분조건
calculate_ltv_limit(50000, "비규제", "1주택처분")
# → LTV 70% = 3.5억 (절대한도 미적용)

# 선순위채권 있는 경우
calculate_ltv_limit(70000, "투기과열", "무주택", senior_liens=5000)
# → 7억 × 40% - 5천만 = 2.3억
```

## 규제지역 목록 (2025.10.15 기준)

**투기과열지구 + 조정대상지역 동시 지정:**
- 서울: 전 25개 자치구
- 경기: 과천, 광명, 성남(분당·수정·중원), 수원(영통·장안·팔달), 안양(동안), 용인(수지), 의왕, 하남

## 생애최초 특례 조건

1. 부부 모두 과거 주택 소유 이력 없음
2. 혼인 여부 무관 (단독세대주 가능)
3. 취득 후 6개월 이내 전입 의무
4. LTV 70% (규제) / 80% (비규제) 적용
5. 정책대출(디딤돌): 부부합산 연소득 7,000만원 이하, 최대 2.4억원
