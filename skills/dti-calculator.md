# DTI (총부채상환비율) 계산기

신DTI 산정기준을 반영하여 주택담보대출의 DTI를 계산하고 대출 한도를 산출합니다.

## DTI vs DSR 차이

| 구분 | DTI (신DTI) | DSR |
|------|------------|-----|
| **분자** | 해당 주담대 원리금 + 기존 주담대 원리금 + 기타대출 **이자만** | **모든** 대출 원리금 전부 |
| **적용** | 주택담보대출 심사 (보조 기준) | 모든 가계대출 (주 기준) |
| **스트레스 가산** | 없음 | 있음 |

> DTI는 기타대출의 **이자만** 포함하는 반면, DSR은 **원리금 전체**를 포함하여 더 엄격합니다.
> 현행 실무에서는 DSR이 주된 기준이며, DTI는 정책대출(디딤돌 등)에 병행 적용됩니다.

## 산정 공식

```
신DTI(%) = (A + B + C) / 연소득 × 100

A = 해당 주담대 연간 원리금 상환액
B = 기존 주담대 연간 원리금 상환액
C = 기타대출(신용·카드론 등) 연간 이자 상환액  ← DSR과의 핵심 차이
```

### 구DTI와 신DTI 차이
- **구DTI**: 기존 주담대도 **이자만** 포함
- **신DTI** (현행): 기존 주담대는 **원리금** 포함, 기타대출은 **이자만** 포함

## DTI 한도

| 지역 | DTI 한도 | 비고 |
|------|:-------:|------|
| 투기지역·투기과열지구 | **40%** | 서울 전역 + 경기 12개 지역 |
| 조정대상지역 | **50%** | |
| 비규제지역 | **60%** | |
| 서민·실수요자 | **60%** | 연소득 7천만원 이하 무주택자, 지역 불문 |
| 정책대출 (디딤돌) | **60%** | |

## 주요 기능

### 1. DTI 계산

**필수 입력**:
- `annual_income`: 연소득 (만원 단위)
- `new_loan_amount`: 신규 주담대 원금 (만원 단위)
- `new_loan_rate`: 약정 연이자율 (%)
- `new_loan_months`: 상환기간 (개월)

**선택 입력**:
- `new_loan_method`: 상환방식 (원리금균등/원금균등/만기일시, 기본: 원리금균등)
- `region`: 지역 (투기과열/조정대상/비규제, 기본: 투기과열)
- `existing_mortgages`: 기존 주담대 목록 (형식: `잔액:금리:잔여개월:상환방식`)
- `other_loans`: 기타대출 목록 (형식: `잔액:금리`) — **이자만 반영**

### 2. DTI 기준 최대 대출 가능액 역산

DTI 한도 내 최대 주담대 금액을 역산합니다.

```
허용 연상환액 = 연소득 × DTI한도율
신규대출 허용 연상환액 = 허용 연상환액 - 기존 주담대 원리금 - 기타대출 이자
최대 대출액 = 역산(신규대출 허용 연상환액, 금리, 만기)
```

## 상환방식별 연간 상환액 산정

- **원리금균등**: M = P × r × (1+r)^n / [(1+r)^n - 1], 연상환 = M × 12
- **원금균등**: 첫해 12개월 합산 (최대 상환액 기준)
- **만기일시**: 연이자 = P × r (원금은 만기 일시상환)

## 사용 예시

```
# 연소득 5천만원, 3억 대출, 금리 4.5%, 30년 원리금균등, 서울(투기과열)
calculate_dti(5000, 30000, 4.5, 360, "원리금균등", "투기과열")

# 기존 주담대 + 신용대출 있을 때
calculate_dti(5000, 30000, 4.5, 360, "원리금균등", "투기과열",
              existing_mortgages="10000:3.5:240:원리금균등",
              other_loans="5000:5.0")
# → 기타대출 5천만원은 이자(250만원/년)만 DTI에 반영

# DTI 기준 최대 대출 가능액
calculate_max_loan_by_dti(5000, 4.5, 360, "원리금균등", "투기과열")
```

## 적용 시 유의사항

1. **DSR과 병행 적용**: 실무에서 DTI와 DSR 모두 통과해야 대출 승인
2. **기타대출 차이**: DTI는 이자만, DSR은 원리금 → DTI가 항상 DSR보다 느슨
3. **정책대출 우대**: 디딤돌 대출은 DTI 60% 적용 (소득 7천만원 이하)
4. **스트레스 미적용**: DTI에는 스트레스 가산금리가 적용되지 않음
