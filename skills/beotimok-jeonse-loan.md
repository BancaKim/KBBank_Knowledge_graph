# 버팀목 전세자금대출 (주택도시기금)

무주택 서민을 위한 전세자금 대출. 일반·청년·신혼 3종으로 구분되며 KB국민은행 등 취급은행에서 신청.

## 3종 비교 요약

| 항목 | 일반 | 청년전용 | 신혼부부 |
|------|------|---------|---------|
| **연령** | 제한 없음 | 만 19~34세 | 혼인 7년 이내 |
| **소득** | 5천만원 | 5천만원 | 7,500만원 |
| **한도(수도권)** | 1.2억 | 1.5억 | 2.2억 |
| **보증금 비율** | 70% | 80% | 80% |
| **금리** | 2.5~3.5% | 2.2~3.3% | 1.9~3.3% |

## 대출기간 · 상환방식

- **기간**: 2년 (임차종료일 초과 불가)
- **연장**: 2년 단위, **최장 10년** 까지
- **상환**: 만기 일시상환 (비대면) / 원리금균등·원금균등 (영업점)

## 주의사항

1. **전입 의무**: 대출 후 1개월 내 전입신고
2. **실거주**: 전입 후 계속 거주 (전출 시 회수 사유)
3. **기금대출 중복 불가**: 디딤돌대출과 동시 이용 불가
4. **보증**: 주택도시보증공사(HUG) 또는 한국주택금융공사(HF) 보증서 필요
5. **보증료**: 연 0.05~0.15% (보증기관별 상이)

## 자격 판별 흐름

```
1. 연령 확인
   → 만 19~34세: 청년전용 우선 검토 (금리 유리)
   → 35세 이상: 일반 또는 신혼 검토

2. 혼인 여부
   → 혼인 7년 이내 / 결혼예정: 신혼부부 검토 (한도·금리 유리)
   → 미혼 34세 이하: 청년전용
   → 미혼 35세 이상: 일반

3. 소득 확인
   → 5천만 이하: 일반·청년 모두 가능 → 금리 비교
   → 5천~7.5천만: 신혼만 가능
   → 7.5천만 초과: 기금대출 불가 → 시중은행 전세대출

4. 출산 2년 이내 여부
   → YES: 신생아 특례 버팀목 검토 (더 유리)
```
