# 대출가능액 종합 계산기

LTV·DTI·DSR 세 가지 규제를 **동시 적용**하여 실제 최대 대출 가능액을 산출합니다.

## 핵심 원칙

```
최종 대출가능액 = MIN(LTV한도, DTI한도, DSR한도, 가격대별 절대한도)
```

> 세 규제 중 **가장 낮은 금액**이 실제 한도입니다. 하나라도 통과하지 못하면 대출이 불가합니다.

## 각 규제별 한도 산정

### 1단계: LTV 한도 (담보가치 기준)

```
LTV한도 = 주택감정가 × LTV율 - 선순위채권 - 소액임차보증금
절대한도 적용 (규제지역): min(LTV한도, 가격대별 상한)
```

→ 상세: [LTV 계산기](ltv-calculator.md)

| 지역·차주 | LTV율 | 절대한도 (15억↓/25억↓/25억↑) |
|----------|:-----:|:------------------------:|
| 규제·무주택 | 40% | 6억 / 4억 / 2억 |
| 규제·생애최초 | 70% | 6억 / 4억 / 2억 |
| 비규제·무주택 | 70% | 미적용 |
| 비규제·생애최초 | 80% | 미적용 |

### 2단계: DTI 한도 (소득 대비 상환비율)

```
허용 연상환액 = 연소득 × DTI한도율
DTI한도 대출액 = 역산(허용 연상환액 - 기존주담대 원리금 - 기타대출 이자)
```

→ 상세: [DTI 계산기](dti-calculator.md)

| 지역 | DTI 한도 |
|------|:-------:|
| 투기과열 | 40% |
| 조정대상 | 50% |
| 비규제 | 60% |

### 3단계: DSR 한도 (전체 부채 원리금 기준)

```
허용 연원리금 = 연소득 × DSR한도율 (은행 40%)
스트레스 적용금리 = 약정금리 + 스트레스가산금리 × 적용비율
DSR한도 대출액 = 역산(허용 연원리금 - 기존 모든 대출 원리금)
```

→ 상세: [DSR 계산기](dsr-calculator.md)

| 금융권 | DSR 한도 | 스트레스 가산 (수도권) |
|-------|:-------:|:------------------:|
| 은행 | 40% | 변동 +3.0%p, 혼합5년 +2.4%p, 고정 0 |
| 2금융 | 50% | 동일 |

### 4단계: 최종 한도 결정

```
최종 = min(LTV한도, DTI한도, DSR한도)
바인딩 제약 = 가장 낮은 한도를 결정한 규제
```

## 주요 기능

### 종합 대출가능액 산출 (`calculate_loan_affordability`)

**필수 입력**:
- `annual_income`: 연소득 (만원)
- `property_value`: 주택 감정가 (만원)
- `loan_rate`: 약정 연이자율 (%)
- `loan_months`: 대출기간 (개월)

**선택 입력**:
- `region_type`: 지역 (투기과열/조정대상/비규제, 기본: 투기과열)
- `borrower_type`: 차주유형 (무주택/생애최초/1주택처분/다주택, 기본: 무주택)
- `repayment_method`: 상환방식 (원리금균등/원금균등/만기일시, 기본: 원리금균등)
- `rate_type`: 금리유형 (변동/혼합5년/혼합10년/고정, 기본: 변동)
- `sector`: 금융권 (은행/2금융, 기본: 은행)
- `senior_liens`: 선순위채권 (만원, 기본: 0)
- `existing_mortgages`: 기존 주담대 (형식: `잔액:금리:잔여개월:상환방식`)
- `other_loans`: 기타대출 (형식: `잔액:금리:잔여개월:상환방식:유형`)

### 출력 내용

```
┌─────────────────────────────────────────────┐
│           대출가능액 종합 분석 결과            │
├──────────┬──────────┬────────────────────────┤
│  규제    │  한도    │  산출 근거              │
├──────────┼──────────┼────────────────────────┤
│  LTV     │  2.8억   │ 7억 × 40%             │
│  DTI     │  3.2억   │ 연소득 6천 × 40%       │
│  DSR     │  2.5억   │ 스트레스 +3.0%p 적용   │
│  절대한도 │  6.0억   │ 15억 이하              │
├──────────┼──────────┼────────────────────────┤
│ ★ 최종   │  2.5억   │ 바인딩: DSR            │
└──────────┴──────────┴────────────────────────┘

▶ 한도 개선 시뮬레이션:
  · 고정금리 전환 시: DSR 2.5억 → 3.1억 (+0.6억)
  · 만기 40년 연장 시: DSR 2.5억 → 2.9억 (+0.4억)
  · 혼합10년 전환 시: DSR 2.5억 → 2.8억 (+0.3억)
```

## 계산 예시

### 예시 1: 서울 7억 아파트, 무주택, 연소득 6천만원

```
calculate_loan_affordability(
    annual_income=6000,
    property_value=70000,
    loan_rate=4.5,
    loan_months=360,
    region_type="투기과열",
    borrower_type="무주택",
    rate_type="변동"
)
```

| 규제 | 한도 | 산출 |
|------|-----:|------|
| LTV | 2.8억 | 7억 × 40% |
| DTI | ~3.2억 | 6천만 × 40% 역산 |
| DSR | ~2.5억 | 스트레스 4.5+3.0=7.5% 적용 |
| **최종** | **~2.5억** | **DSR이 바인딩** |

### 예시 2: 비규제 5억 아파트, 생애최초, 연소득 5천만원

```
calculate_loan_affordability(
    annual_income=5000,
    property_value=50000,
    loan_rate=3.8,
    loan_months=360,
    region_type="비규제",
    borrower_type="생애최초",
    rate_type="고정"
)
```

| 규제 | 한도 | 산출 |
|------|-----:|------|
| LTV | 4.0억 | 5억 × 80% |
| DTI | ~3.2억 | 5천만 × 60% 역산 |
| DSR | ~3.4억 | 고정금리 스트레스 0% |
| **최종** | **~3.2억** | **DTI가 바인딩** |

### 예시 3: 기존 대출이 있는 경우

```
calculate_loan_affordability(
    annual_income=8000,
    property_value=90000,
    loan_rate=4.0,
    loan_months=360,
    region_type="투기과열",
    borrower_type="무주택",
    rate_type="혼합5년",
    existing_mortgages="10000:3.5:240:원리금균등",
    other_loans="3000:5.0:36:원리금균등:신용대출"
)
```

> 기존 대출이 DTI·DSR 여유분을 줄여 최종 한도가 크게 감소합니다.

## 한도 개선 전략 (바인딩 규제별)

### DSR이 바인딩일 때
1. **고정금리 전환** → 스트레스 가산 0% (가장 효과적)
2. **대출기간 연장** → 연상환액 감소 (30년→40년)
3. **기존 대출 상환** → DSR 여유 확보
4. **소득 증빙 보완** → 부부합산소득 활용

### LTV가 바인딩일 때
1. **생애최초 자격 확인** → LTV 40%→70% 대폭 상향
2. **저가 주택 선택** → 절대한도 영향 감소
3. **선순위채권 상환** → LTV 여유 확보

### DTI가 바인딩일 때
1. **기타대출 상환** → DTI는 이자만 반영하므로 효과 제한적
2. **기존 주담대 상환/리파이낸싱** → 원리금 감소
3. **비규제지역 매수** → DTI 60% 적용
