# 한국 부동산·금융 대출 규제 (2026년 3월 기준)

> **최종 업데이트**: 2026-03-25
> **근거**: 금융위원회 6·27 대책(2025.6.27), 10·15 대책(2025.10.15), 스트레스 DSR 3단계(2025.7.1)
> **주의**: 정부 정책에 따라 규제가 수시 변경됩니다. 정확한 내용은 금융위원회/금융감독원 공시를 확인하세요.

## 1. LTV (주택담보인정비율) 규제

### 규제지역 vs 비규제지역 LTV (2025년 6·27 대책 이후)

| 차주 유형 | 규제지역 | 비규제지역 |
|----------|:-------:|:--------:|
| 무주택자 (일반) | **40%** | **70%** |
| **생애최초 주택구입** | **70%** | **80%** |
| 1주택자 (6개월 내 처분 조건) | **50%** | **70%** |
| 1주택자 (처분 조건 없음) | **0% (금지)** | — |
| 다주택자 (추가 구입) | **0% (금지)** | — |

### 주택가격대별 대출 한도 상한 (규제지역, 10·15 대책)

| 주택 시가 | 대출 한도 상한 |
|-----------|:------------:|
| 15억원 이하 | **최대 6억원** |
| 15억 초과 ~ 25억 이하 | **최대 4억원** |
| 25억원 초과 | **최대 2억원** |

> LTV 비율과 한도 상한 중 **낮은 금액**이 실제 대출 가능액.
> 9억원 기준 LTV 차등은 현행 제도에서 **폐지**됨. 현재는 15억/25억 기준.

### 생애최초 주택구입자 특례

| 항목 | 내용 |
|------|------|
| LTV | 규제지역 **70%**, 비규제지역 80% |
| 한도 상한 | **최대 6억원** |
| 전입 의무 | **6개월 이내** 실거주 전입 |
| 대상 | 세대원 전원 무주택, 생애 최초 주택 구입 |

**계산 예시 (규제지역, 9억 주택, 생애최초)**:
- LTV 70%: 9억 × 70% = **6.3억**
- 한도 상한: **6억**
- **실제 대출 가능: 6억** (한도 상한 적용)

## 2. DSR (총부채원리금상환비율) 규제

### DSR 기본 한도

| 금융권 | DSR 한도 | 적용 기준 |
|--------|:-------:|----------|
| 은행권 | **40%** | 총 대출 잔액 1억 초과 |
| 2금융권 | **50%** | 총 대출 잔액 1억 초과 |

### 스트레스 DSR (3단계, 2025.7.1 시행)

| 구분 | 스트레스 가산금리 |
|------|:---------------:|
| 수도권·규제지역 주담대 | **+3.0%p** (10·15 대책 추가 인상) |
| 지방 주담대 | **+0.75%p** |
| 신용대출 (1억 초과) | **+1.5%p** |

### DSR 계산 방법

```
DSR = (모든 금융부채의 연간 원리금 상환액) / 연소득 × 100

⚠️ 소득 정보 없이는 DSR 기반 대출가능액을 계산할 수 없음.
반드시 고객에게 연소득(부부합산)과 기존 대출 잔액을 확인 후 계산할 것.
```

### DSR 산정 시 포함 부채

| 포함 | 미포함 |
|------|--------|
| 주택담보대출 원리금 | 보증부 전세대출 (일부 예외) |
| 신용대출 원리금 | 학자금 대출 (ICL 방식) |
| 자동차할부/리스 | 500만원 이하 소액 대출 |
| 카드론/현금서비스 | — |
| 기존 대출 잔액의 원리금 | — |

## 7. 답변 시 필수 규칙

### 반드시 고객에게 확인할 정보

대출 한도 질문 시 다음을 **먼저 확인**하고 답변할 것:

1. **주택 소재지**: 규제지역/비규제지역
2. **주택 가격**: 15억 이하/초과
3. **생애최초 여부**: 생애 최초 주택 구입인지
4. **연소득**: DSR 계산에 필수 (부부합산)
5. **기존 부채**: 다른 대출이 있는지
6. **주택 보유 현황**: 무주택/1주택/다주택

### 소득 정보 없을 때

```
⚠️ 소득 정보 없이는 DSR 기반 대출 가능액을 정확히 계산할 수 없습니다.
LTV 기준 대출 가능액만 안내하고, DSR은 소득 확인 후 계산해드리겠다고 안내하세요.
"정확한 대출 가능 금액을 안내해드리기 위해 연소득(부부합산)을 알려주시겠어요?"
```

### 답변 템플릿

```
[주택가격] X억원 / [지역] 규제지역·비규제지역 / [생애최초] 여부

■ LTV 기준 대출 가능액
  - LTV X%: X억 × X% = X억원
  - 한도 상한: X억원
  - → LTV 기준: X억원

■ DSR 기준 대출 가능액 (소득 확인 필요)
  - 연소득 X만원 기준 / DSR 40% / 스트레스 금리 +3.0%p 반영
  - → DSR 기준: 약 X억원

■ 실질 대출 가능액
  - LTV와 DSR 중 낮은 금액: 약 X억원

※ 정확한 대출 가능 금액은 개인 신용도, 소득 증빙, 기존 부채를
   종합 심사하여 결정됩니다.
※ KB국민은행 고객센터(1588-9999) 또는 영업점 상담을 권장합니다.
```
