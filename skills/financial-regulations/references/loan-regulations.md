# 한국 부동산·금융 대출 규제 (2026년 기준)

> **최종 업데이트**: 2026-03-25
> **주의**: 정부 정책에 따라 규제가 수시 변경됩니다. 정확한 내용은 금융위원회/금융감독원 공시를 확인하세요.

---

## 1. LTV (Loan-to-Value) 규제

### 주택담보대출 LTV 한도

| 구분 | 규제지역 | 비규제지역 |
|------|:-------:|:--------:|
| 일반 (무주택/1주택) | **50%** | **70%** |
| 생애최초 주택구입 | **80%** | **80%** |
| 2주택 이상 | **30%** | **60%** |
| 투기지역/투기과열지구 | **40%** | — |

### 주택가격대별 대출 한도 (규제지역 기준)

| 주택가격 | LTV 적용 | 실질 최대 대출 |
|----------|---------|:------------:|
| 9억원 이하 | 50% | **최대 6억원** 아님 — 주택가격 × LTV |
| 9억원 초과~15억원 | 9억까지 50% + 초과분 30% | **계산 필요** |
| 15억원 초과 | 대출 불가 (규제지역) | **0원** |

**계산 예시 (규제지역, 12억 주택)**:
- 9억 × 50% = 4.5억
- 3억(초과분) × 30% = 0.9억
- **합계: 5.4억**

### 생애최초 주택구입자 특례

| 조건 | 내용 |
|------|------|
| 대상 | 세대원 전원 무주택, 생애 최초 주택 구입 |
| LTV | **80%** (규제지역 포함) |
| 주택가격 | 9억원 이하 |
| 소득요건 | 부부합산 연소득 1억원 이하 (2024년 기준, 변동 가능) |
| 한도 | **최대 6억원** |

## 2. DSR (Debt Service Ratio) 규제

### DSR 기준

| 대출 유형 | DSR 한도 |
|----------|:-------:|
| 은행권 주택담보대출 | **40%** |
| 2금융권 주택담보대출 | **50%** |
| 신용대출 (1억 초과) | **40%** |
| 신용대출 (1억 이하) | DSR 미적용 (단, 총부채 기준 관리) |

### DSR 계산 방법

```
DSR = (모든 금융부채의 연간 원리금 상환액) / 연소득 × 100

예시:
- 연소득: 6,000만원
- DSR 40% 적용 시: 연간 상환가능액 = 2,400만원 (월 200만원)
- 연 4% 금리, 30년 상환 시: 대출가능액 약 4.2억원
```

### DSR 산정 시 포함 부채

| 포함 | 미포함 |
|------|--------|
| 주택담보대출 원리금 | 전세보증금 반환 대출 (일부 예외) |
| 신용대출 원리금 | 학자금 대출 (ICL 방식) |
| 자동차할부 원리금 | 500만원 이하 소액 대출 |
| 카드론/현금서비스 | — |
| 기존 대출 잔액의 원리금 | — |

## 3. 대출 총량 규제

### 개인별 총 대출한도

| 구분 | 내용 |
|------|------|
| 차주단위 DSR | 모든 금융권 대출의 DSR을 통합 관리 |
| 스트레스 DSR | 금리 상승 시나리오(+1.5~3.0%p) 반영하여 산정 |
| 은행별 가계대출 총량 관리 | 금융당국이 은행별 가계대출 증가율 관리 |

### 스트레스 DSR

```
스트레스 금리 = 현재 금리 + 스트레스 가산금리(1.5~3.0%p)

예시:
- 현재 금리 4.0% + 스트레스 2.0%p = 6.0%로 DSR 산정
- 실제 대출가능액이 일반 DSR 대비 약 15~25% 감소
```

## 4. 주택담보대출 유형별 규제

### 4a. 일반 주택담보대출

| 항목 | 내용 |
|------|------|
| LTV | 규제지역 50%, 비규제지역 70% |
| DSR | 40% (은행권) |
| 상환방법 | 거치기간 1년 이내 권장, 분할상환 원칙 |
| 대출기간 | 최장 30~40년 (은행별 상이) |
| 중도상환수수료 | 대출 후 3년 이내 상환 시 1.2~1.4% |

### 4b. 전세자금대출

| 항목 | 내용 |
|------|------|
| 한도 | 전세보증금의 **80%** 이내 (최대 수도권 5억, 비수도권 3억) |
| 보증기관 | SGI서울보증, HF한국주택금융공사, HUG주택도시보증공사 |
| 보증료 | 연 0.02~0.15% (보증기관·신용도별) |
| DSR | 비적용 (보증부 전세대출은 DSR 제외) |
| 갱신 | 최초 2년, 연장 시 2년 단위 |
| 소득요건 | 부부합산 연소득 1억원 이하 (보증 유형별 상이) |

### 4c. 신용대출

| 항목 | 내용 |
|------|------|
| 한도 | 연소득 이내 원칙 |
| DSR | 1억원 초과 시 40% 적용 |
| 금리 | 개인 신용등급(CSS/NICE) 기반 차등 |
| 기간 | 최장 5~10년 |
| 주택구입 목적 제한 | 규제지역 주택구입 목적 신용대출 제한 |

### 4d. 자동차대출

| 항목 | 내용 |
|------|------|
| 한도 | 차량가격의 70~80% |
| DSR | 포함 (원리금 상환액 반영) |
| 기간 | 최장 5~7년 (거치 없음이 일반적) |
| 담보 | 차량 저당권 설정 |

## 7. 답변 시 필수 안내 사항

대출 한도/금액 관련 질문에 답변할 때 반드시 포함해야 할 내용:

1. **LTV 규제**: 규제지역/비규제지역 여부에 따른 LTV 차이
2. **DSR 규제**: 연소득 대비 총 상환능력 기준
3. **주택가격대별 한도**: 9억/15억 기준 차등 적용
4. **생애최초 여부**: 생애최초 구입자는 LTV 80% 특례
5. **다주택자 여부**: 2주택 이상 시 LTV 30% (규제지역)
6. **기존 부채 반영**: DSR 산정 시 기존 대출 포함
7. **스트레스 DSR**: 금리 상승 시나리오 반영
8. **정확한 금액은 개인 심사 필요**: "정확한 대출 가능 금액은 소득, 신용도, 기존 부채 등을 종합적으로 심사하여 결정됩니다"

### 답변 템플릿

```
[주택가격] X억원, [지역] 규제지역/비규제지역, [생애최초] 여부 기준:

■ LTV 기준 대출 가능액
  - LTV X%: 최대 X억원

■ DSR 기준 대출 가능액
  - 연소득 X만원, DSR 40% 기준: 최대 약 X억원
  - (기존 부채가 있으면 추가 차감)

■ 실질 대출 가능액
  - LTV와 DSR 중 낮은 금액 적용: 약 X억원

※ 정확한 대출 가능 금액은 개인 신용도, 소득 증빙, 기존 부채 등을
   종합적으로 심사하여 결정됩니다. KB국민은행 고객센터(1588-9999)
   또는 가까운 영업점에서 상세 상담을 받으실 수 있습니다.
```
