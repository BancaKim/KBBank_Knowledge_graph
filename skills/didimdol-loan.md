# 디딤돌대출 (주택도시기금)

무주택 서민을 위한 저금리 주택구입자금 대출. 소득구간별 차등금리 적용, 우대금리 다양.

## 자격요건

| 항목 | 조건 |
|------|------|
| **소득** | 부부합산 연소득 6천만원 이하 |
| | 생애최초·2자녀 이상: 7천만원 이하 |
| | 신혼가구: 8,500만원 이하 |
| **자산** | 순자산가액 5.11억원 이하 (2026년 기준) |
| **주택가격** | 5억원 이하 (신혼·2자녀 이상: 6억원) |
| **면적** | 전용 85㎡ 이하 (읍·면 100㎡) |
| **주택보유** | 세대원 전원 무주택 |
| **세대주** | 대출접수일 현재 세대주 |
| **신용** | CB점수 350점 이상 |

## 대출한도

| 차주 유형 | 최대 한도 |
|----------|:--------:|
| 일반 | **2.0억원** |
| 생애최초 | **2.4억원** |
| 신혼·2자녀 이상 | **3.2억원** |

> LTV·DTI 범위 내에서 위 한도 적용

### 추가 우대 (중복 적용 가능)

| 우대 항목 | 할인폭 | 조건 |
|----------|:------:|------|
| 청약저축 (5년+) | **0.5%p** | 가입 5년 이상 |
| 청약저축 (3~5년) | **0.3%p** | 가입 3~5년 |
| 전자계약 | **0.1%p** | 부동산 전자계약 체결 |
| 저금액 신청 | **0.1%p** | 대출가능액의 30% 이하 신청 |
| 중도상환 | **0.2%p** | 원금 40% 이상 상환 후 재대출 |

### 금리 계산 예시

```
연소득 4천만원, 30년, 생애최초, 청약저축 5년:
기본금리 3.45% - 생애최초 0.2% - 청약저축 0.5% = 2.75%

연소득 3천만원, 신혼, 20년, 1자녀, 전자계약:
신혼금리 3.10% - 1자녀 0.3% - 전자계약 0.1% = 2.70%
```

## 대출기간 · 상환방식

- **기간**: 10, 15, 20, 30년
- **상환방식**: 원리금균등 / 원금균등 / 체증식 분할상환
- **거치기간**: 1년 또는 비거치
- **금리유형**: 고정금리 또는 5년 단위 변동금리

## 주의사항

1. **실거주 의무**: 대출 후 1개월 내 전입, 1년 이상 실거주
2. **중도상환수수료**: 대출실행 후 3년 이내 0.5%
3. **기금대출 중복 불가**: 디딤돌+버팀목 동시 이용 불가
4. **전세대출 전환**: 기존 전세대출 상환 후 디딤돌 전환 가능
