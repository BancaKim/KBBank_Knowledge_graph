# 보금자리론 (한국주택금융공사)

장기 고정금리 주택담보대출. 한국주택금융공사(HF)가 유동화하여 금리 변동 위험 없이 최대 50년까지 상환 가능.

## 자격요건

| 항목 | 조건 |
|------|------|
| **소득** | 부부합산 연소득 7천만원 이하 |
| | 신혼부부: 8,500만원 이하 |
| | 다자녀(2자녀): 8,000만원 이하 |
| | 다자녀(3자녀+): 1억원 이하 |
| **주택가격** | 6억원 이하 (공부상 주택) |
| **주택보유** | 무주택 또는 1주택 (대환·이주 목적) |
| **연령** | 민법상 성년 |
| **국적** | 대한민국 국민 (재외국민·동포 포함) |
| **신용** | CB점수 271점 이상, 연체·부도 이력 없음 |

## 대출한도

| 차주 유형 | 최대 한도 |
|----------|:--------:|
| 일반 | **3.6억원** |
| 다자녀·전세사기피해자 | **4.0억원** |
| 생애최초 | **4.2억원** |

## 우대금리 (최대 1.0%p, 중복 적용 가능)

| 우대 항목 | 할인폭 | 조건 |
|----------|:------:|------|
| 전자약정·전자등기 | 0.1%p | 아낌e 이용 시 |
| 저소득청년 | 0.1%p | 만 34세 이하, 소득 7천만원 이하 |
| 신혼가구 | 0.3%p | 혼인 7년 이내 |
| 신생아 출산가구 | 0.2%p | 2년 내 출산 |
| 2자녀 가구 | 0.5%p | |
| 3자녀 이상 | 0.7%p | |
| 한부모·장애인·다문화 | 0.7%p | |
| 전세사기피해자 | 1.0%p | 확인서 필요 |

> 우대금리 적용 시 최저 금리: 약 **2.90%~3.20%** (10~50년)

## 대출기간 · 상환방식

- **기간**: 10, 15, 20, 30, 40, 50년
- **상환방식**: 원리금균등 / 원금균등 / 체증식 분할상환
- **거치기간**: 1년 또는 비거치

## 중도상환수수료

- 3년 이내: 대출잔액의 **0.5%** 한도
- 3년 초과: 면제
- 면제 대상: 저소득청년, 다자녀, 전세사기피해자 등

## 디딤돌대출과 비교

| 항목 | 보금자리론 | 디딤돌대출 |
|------|----------|----------|
| 운영기관 | 한국주택금융공사(HF) | 주택도시기금 |
| 소득기준 | 7천만원 | 6천만원 (신혼 8.5천만) |
| 최대한도 | 4.2억 (생애최초) | 4억 (신혼·다자녀) |
| 주택가격 | 6억원 이하 | 5억원 이하 (신혼 6억) |
| 금리 | 4.05~4.45% (고정) | 2.65~3.80% (소득별) |
| 금리유형 | **순수 고정금리** | 고정 또는 5년 변동 |
| 최장기간 | 50년 | 30년 |

> **보금자리론**: 금리는 높지만 순수 고정금리 + 최대 50년 → 금리 상승기에 유리
> **디딤돌**: 금리 낮지만 소득·주택가격 조건 엄격
